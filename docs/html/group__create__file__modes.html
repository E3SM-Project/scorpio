Supported modes for creating a file

: The flags to specify the file access mode when creating a file PIO_CLOBBER : The file, if it exists, is truncated (the old contents of the file are deleted/overwritten) PIO_NOCLOBBER : This flag is used to ensure that a file, if it exists, is not truncated (the current contents of the file are not deleted) during file creation.
