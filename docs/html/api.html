SCORPIO user interface

The user API is closely related to the NetCDF API. Please refer to http://www.unidata.ucar.edu/software/netcdf/docs for more information on NetCDF APIs. The SCORPIO API is listed below:

Routines for file operations

  • PIO_openfile
  • PIO_createfile
  • PIO_syncfile
  • PIO_closefile

Library init/finalize routines

  • PIO_init
  • PIO_finalize

I/O decomposition routines

  • PIO_initdecomp
  • PIO_freedecomp

Reading and Writing distributed variables

  • PIO_read_darray
  • PIO_write_darray

Misc Utility routines

  • PIO_set_hint
  • PIO_setframe
  • PIO_advanceframe
  • PIO_setdebuglevel
  • PIO_seterrorhandling
  • PIO_get_local_array_size
  • PIO_getnumiotasks
  • PIO_set_blocksize

Non-distributed data routines

Reading/Writing metadata

  • PIO_get_att
  • PIO_put_att
  • PIO_get_var
  • PIO_put_var

Routines for defining variables and metadata

  • PIO_def_var
